## Currency Converter

Projeto simples de convertor de moedas pulicado pelo github pages.

## Site publicado

https://arissa-inada.github.io/Currency-Converter/

## Sobre o projeto

Este projeto foi criado para praticar:

Estruturação com HTML

Estilização com CSS

Manipulação de lógica com JavaScript

Consumo de API de câmbio (Exchange Rates)

Publicação com GitHub Pages

A aplicação converte valores entre diferentes moedas com base em dados atualizados.

🛠 Tecnologias usadas

HTML5

CSS3

JavaScript
